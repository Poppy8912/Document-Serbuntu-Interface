# Serbuntu
A lightweight cloud-style document interface built with HTML, CSS, and JavaScript.

Serbuntu is a browser-based document management UI inspired by modern cloud storage systems. The project focuses on interactive frontend behavior, clean UI design, local storage functionality, and dynamic document management.

Features

Document Management
- Create new documents dynamically
- Expand/collapse document sections
- Delete documents into a trash system
- Restore deleted documents from trash
- Save document contents using localStorage

Interactive UI
- Custom right-click context menu
- Double-click popup creation menu
- Responsive flexbox layout
- Animated hover effects
- Dark themed clooud-style interface

User System
- Displays logged-in user from localStorage
- Toggleable profile popup
- Persistent user display across reloads

Simulated Server Connection
- Placeholder server connection flow
- Connection status badge
- URL validation system
- Dynamic connection state updaets

Technologies Uesd
- HTML5
- CSS3
- JavaScript 
- LocalStorage API
- Flexbox Layout
- DOM Manipulation

 This project was built to strength:
    - Frontend development skills
    - Dynamic DOM manipulation

    - Interactive application structure
